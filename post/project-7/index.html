Project VII: School Transportation Data Dashboard

October 17, 2024

I have successfully developed and deployed an Interactive School Transportation Data Dashboard that offers real-time insights into key performance indicators, including estimated and actual fuel consumption for school vans on a weekly basis. This dashboard helps schools optimize their transportation efficiency. My contributions and responsibilities for this project included:

Project Highlights:

  • Data Collection & Structuring
    I sourced transportation data, including fuel consumption and route performance, and structured it using Google Sheets integrated with MySQL for storage.

  • Exploratory Data Analysis (EDA)
    Performed comprehensive EDA to identify trends and gain insights such as comparing fuel estimates against actual usage. Data visualizations were developed using Looker studio and Excel.

  • Data Cleaning & Preprocessing
    Used excel and excel pivot tables to clean the dataset by removing null values and ensuring the data was ready for accurate analysis.

  • Dashboard Design & Implementation
    Designed an intuitive dashboard using Google Looker Studio with live data from Google Sheets. The dashboard includes interactive charts and graphs that help school administrators track van performance and fuel usage in real-time.

  • Web Integration & Deployment
    Successfully hosted the dashboard on an HTML website and integrated it into a WordPress site using Elementor, enhancing the user experience for both administrators and school clients.

  • Client Communication & Outreach
    Launched an email marketing campaign introducing the dashboard to various schools, explaining its benefits in reducing transportation costs and improving efficiency.
